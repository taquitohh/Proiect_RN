# Screenshots Folder

Acest folder conține capturi de ecran demonstrative pentru interfața UI.

## Fișiere necesare:
- `ui_demo.png` - Screenshot cu interfața de chat funcțională

## Cum să faci screenshot:
1. Pornește backend-ul: `cd src && python api.py`
2. Pornește frontend-ul: `cd frontend && npm run dev`
3. Deschide http://localhost:5173 în browser
4. Introduce o comandă (ex: "creează un cub roșu")
5. Fă screenshot după ce apare codul generat
6. Salvează ca `ui_demo.png` în acest folder
